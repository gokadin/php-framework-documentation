Validation
# Usage in controllers
The validator object can validate an array of data based on a set of rules, returning a ValidatorResult object containing the success state and a list or errors if any. It can be accessed from the validator property in controllers.
App/Http/Controllers/UserController.php
public function fetchUserPost()
{
    $result = $this->validator->validate($this->request->all(), [
        'id' => 'min:1', // single validation
        'postId' => ['numeric', 'min:1'], // multiple validations
    ]);

    if (!$result->isValid())
    {
        return $this->respondBadRequest($result->errors());
    }

    return $this->respondOk();
}
You can also use the helper method validate in the controller, which automatically use the data from the request.
App/Http/Controllers/UserController.php
public function fetchUserPost()
{
    $result = $this->validate([
        'id' => 'min:1',
    ]);

    // ...
}
